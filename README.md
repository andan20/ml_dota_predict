# ml_dota_predict
ML HSE course homework. The task was  to build а linear model that predicts the winner of a Dota 2 match.


# Идея
По разным данным о матчах (выбранные герои, регион сервера матча, график преимущества по опыту/золоту в первые минуты для некоторых матчей и т.д.) предсказать команду победителя матча. Метрика - коэф Джини, соревнование на kaggle.com .

# Итог
К моменту дедлайна занимал 40 место из 400 участников, получив значение 0.383 коэффициента Джини, успешно преодолев требуемые авторами трешхолды в 0.24 и 0.26. 

# Ссылка на соревнование
https://www.kaggle.com/competitions/dota-2-hse-ml-1-course-competition-2026/overview

# Используемые технологии
Pandas, matplotlib, sklearn, numpy, optuna(для подбора гиперпараметров лог рега).

# Что применял в проекте
Визуализации данных, различные энкодинги, кросс-валидация, ROC-AUC метрика (для вычисления коэф-та Джини), агрегации данных, логистическая регрессия, скейлинг данных. Создан собственный пайплан для удобного запуска и теста моделей.
